# Vạn Xuân Lab

> Nghiên cứu phần mềm hệ thống và trí tuệ nhân tạo từ Việt Nam.

Vạn Xuân Lab tập trung vào các lĩnh vực sâu trong máy tính:
ngôn ngữ lập trình, compiler, hệ điều hành, và AI.  
Chúng tôi xây dựng những thứ chạy bên dưới, để mọi thứ bên trên hoạt động tốt hơn.

---

## Lĩnh vực nghiên cứu

**🔧 Hệ thống phần mềm**  
Ngôn ngữ lập trình, compiler, runtime, toolchain.

**🤖 Trí tuệ nhân tạo**  
Mô hình ngôn ngữ, kiến trúc AI, nghiên cứu ứng dụng thực tế.

---

## Dự án

### 🔷 [Draton](https://github.com/draton-lang/draton)
Ngôn ngữ biên dịch, kiểu tĩnh, LLVM backend.  
Type inference mạnh, toolchain tích hợp (`drat`).

### 🟩 [Thagore](https://github.com/thagore-foundation/thagore)
Ngôn ngữ nhẹ, indentation-based — viết như đang suy nghĩ.  
VM/WASM/Cranelift backend, package manager `drago`.

### 🧠 [Vietrix](https://github.com/vietrix)
Nghiên cứu AI — mô hình ngôn ngữ và kiến trúc học sâu.
